Dashboard einrichten

Verbinde das Dashboard einmalig mit deiner Notion-Datenbank. Die Daten werden dann live aus Notion geladen.

So bekommst du deinen API Key:
1. Geh auf notion.so/my-integrations
2. „New integration" → Name: Dashboard → Submit
3. API Key kopieren (beginnt mit secret_)
4. In Notion: Datenbank öffnen → „..." → Connections → Integration hinzufügen
Die ID findest du in der URL deiner Notion-Datenbank: notion.so/DIESE-ID-HIER?v=...
